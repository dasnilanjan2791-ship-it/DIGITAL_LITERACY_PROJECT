## 🔐 TASK 5: Phishing Awareness

### ⚠️ What is Phishing?
A cyberattack where attackers trick users into revealing sensitive information.

### 🧠 How It Works:
1. Fake message from trusted source  
2. Urgent call-to-action  
3. Redirect to fake website  
4. User enters sensitive data  

### 🚨 Risks:
- Financial loss  
- Identity theft  
- Account compromise  

### 🛡️ Prevention Checklist:
- Verify sender email  
- Never share OTP/passwords  
- Use strong passwords  
- Enable two-factor authentication  
- Avoid suspicious links  
- Check URLs carefully  
- Avoid public Wi-Fi for transactions  
- Keep apps updated  

### 💳 UPI Safety:
- Do not approve unknown requests  
- Verify receiver before sending money  

### 📞 Emergency:
- Report fraud: https://cybercrime.gov.in  
- Call: **1930**

### 💡 Learning Outcome:
- Realized how convincing phishing attacks can be  
- Developed habit of verifying links and sources
